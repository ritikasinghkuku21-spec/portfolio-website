# Personal Portfolio Website

A responsive multi-page personal portfolio website built using HTML5, CSS3, and JavaScript. The website showcases my skills, projects, and contact information while following modern web development practices.

## ✨ Features

- Responsive design for desktop and mobile devices
- Semantic HTML5 structure
- Clean and modern user interface
- About Me section
- Projects showcase
- Contact page with accessible form
- SEO-friendly meta tags
- Accessibility using ARIA labels

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
portfolio/

│── projects.html
│── contact.html
│── assets/

│   ├── main.js

```

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/portfolio-website.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser or run it using Live Server.
